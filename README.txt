Le projet consiste à utiliser les fonctions, comme git push, git add, git commit et pouvoir partager sur plusieurs remotes les informtions des fichiers d'un répertoire.
Je dois ainsi creer 3 fichiers, avec le fichier readme.md qui est cree à partir de la cration du repository sur github.
ces 3 fichiers seront modifié et suivis par des commits.
tout ceci doit être partagé dur github et sur l'ordinateur par l'intermédiaire de la console git bash.
Vim permet de modifier le contenu des fichiers.
